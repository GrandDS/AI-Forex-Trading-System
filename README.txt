AI Forex Trading System

Description:
A modular demonstration trading system that simulates market data, signal generation, and trade execution.

Files:
- config.py: configuration values
- data.py: market data simulation
- strategy.py: placeholder decision logic
- execution.py: trade execution and balance tracking
- logger.py: simple logging helper
- main.py: orchestration script

Run:
python main.py

Note:
This project is portfolio-safe and uses simplified placeholder logic rather than proprietary strategy code.
